Roller Derby Quiz App

This is a multi-page quiz app that tracks a user’s score on each page. The user is answering questions related to roller derby. As the user makes their way through the app, they either receive a “you fell down” if they get the question wrong or a “jam on!” if they answer the question correctly. At the end of the app, the user has the option to try again.
